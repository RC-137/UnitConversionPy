# UnitConversionPy

A lightweight **desktop unit converter** built with **Python** and **PyWebView**, featuring a modern HTML/CSS interface. Supports **Length** and **Temperature** conversions and is easily expandable to other unit categories.

---

## Features
  
- Convert **length units**: mm, cm, m, dm, hm, km, in, ft, yd, and mi
- Convert **temperature units**: °C, °F, K  
- Single-page interface with dynamic dropdowns  
- Modern, responsive design  
- Unlimited conversions without restarting the app  
- Easy to extend for other unit categories like Weight, Time, Speed
